stdlibx-option
==============

.. toctree::
   :maxdepth: 1
   :hidden:

   Getting Started <self>
   Reference <reference/stdlibx.option>

This package provides an option type that represents either a value (:py:class:`~stdlibx.option.Some`) or the absence of a value (:py:class:`~stdlibx.option.Nothing`). It offers a safer and more explicit alternative to None, making it easier to handle optional values in a composable and predictable way.

------------
